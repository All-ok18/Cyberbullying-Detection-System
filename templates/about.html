About Cyberbullying Detection

This project uses Machine Learning and Flask to predict whether a sentence contains cyberbullying or non-bullying content.

⚠ Cyberbullying

  • Insults
  • Abusive language
  • Threatening messages
  • Harassment
  • Negative targeting
  • Offensive comments
  • Repeated harmful behavior

✅ Non Bullying

  • Positive messages
  • Friendly communication
  • Normal opinions
  • Constructive feedback
  • Respectful language
  • Motivational text
  • Healthy discussion
